**Internal Processing:**

- Simulates 5+ scenarios simultaneously

**External Output:**
- Sequential output channels (speech, writing, typing) create bottleneck
- Translation from parallel thought to linear expression is costly
- Hands/words cannot keep pace with mental processing
- "Knows" answers but struggles to "show work"

### Learning Sequence Requirement

```
ALWAYS: Concept → Logic → Math → Code → Application

NEVER: Syntax → Memorize → Practice → Maybe understand later
```

- Stores relationships between concepts, not isolated facts
- Retains patterns, not procedures
- Remembers flow, not syntax
- Strong: "Dot product measures similarity"
